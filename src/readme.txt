Cara Menjalankan:
1. Pastikan anda sudah install Python minimal versi 3.8.
2. Download yt-dlp.exe dari: 
https://github.com/yt-dlp/yt-dlp/releases
3. Download ffmpeg.exe dari: 
https://ffmpeg.org/download.html
4. letakan file tersebut sejajar dengan file main.py
5. Jalankan program ini dengan:
    python main.py

Keterangan:
- Program ini hanya interface sederhana untuk menggunakan yt-dlp dan ffmpeg.
- Anda  bisa mengubah lokasi yt-dlp dan ffmpeg di dalam kode `main.py` sesuai kebutuhan.
